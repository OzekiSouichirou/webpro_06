# エイトNOW ドキュメント

## 概要

**エイトNOW** は、Node.js の標準モジュール（`http`, `fs`, `path`）を用いて構築した簡易的なウェブアプリケーションである。  
シングルページアプリケーション（SPA）の仕組みを用いて在庫情報を表示し、購入や在庫リセットなどの操作を行えるようになっている。  
フレームワーク（例: Express）や外部ライブラリを一切使用せず、授業で扱った技術だけで構成されている。
